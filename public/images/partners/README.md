# Partner Logos

Place partner logo images in this directory.

## Required Files

- `partner-1.png`
- `partner-2.png`
- `partner-3.png`
- `partner-4.png`

## Image Specifications

- Format: PNG (with transparency) or SVG
- Recommended size: 180px width (height will be auto-adjusted)
- Aspect ratio: Flexible (logos will be contained within the allocated space)
- Background: Transparent preferred

## Usage

Logos are displayed in the "Our Trusted Partners" section on the homepage.

The component automatically handles:
- Responsive sizing (2x2 grid on mobile, 4 in a row on desktop)
- Hover effects (opacity transition)
- Lazy loading
