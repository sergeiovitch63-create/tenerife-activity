# Flyers Directory

This directory contains flyer images for the partner section.

## Partner Flyer

- **Recommended filename**: `partner-flyer-a5.png` (or `.jpg`)
- **Recommended resolution**: Print-quality / high-res
- **Format**: A5

## Usage

The site will load the flyer image from:
```
/flyers/partner-flyer-a5.png
```

Make sure the image is optimized for web while maintaining print quality.
